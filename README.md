# Summary

Old English [Cairo](https://github.com/UniversalDependencies/cairo) sentences with UD and additional annotations


# Introduction

Old English [Cairo](https://github.com/UniversalDependencies/cairo) sentences with UD and additional annotations
relevant to its historicity like hyperlemma, Indo-European root, and gloss.
